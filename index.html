HTML & CSS Piscine

About

42 HTML & CSS Piscine 에 오신 여러분을 환영합니다 🐣

HTML & CSS Piscine은 7일 동안 동료들과 함께 토이 프로젝트들을 수행하며 웹페이지의 뼈대를 이루는 HTML & CSS 기술을 익히는 프로젝트입니다.

여러분은 아래의 기법들을 직접 사용해보며 다양한 디자인의 웹페이지들을 구현할 것 입니다.

  • HTML tags & semantics
  • CSS syntax
  • CSS Layout (block, flexbox, grid, etc.)
  • CSS decoration
  • CSS animation
  • Responsive web design

Piscine 에서 살아남을 수 있는 방법은 단 하나 뿐이죠. 모르는게 있으신가요? 양 옆에 있는 동료에게 물어보세요. 양 옆에 있는 동료도 모른다면 세명이 함께 다른 동료를 찾아가 질문하세요. Google 이 여러분과 함께합니다.

HTML & CSS 피신에서는 clean coding을 위해 최소한의 지침을 제공합니다. 과제 수행 시 지침을 꼭 지켜주세요.
