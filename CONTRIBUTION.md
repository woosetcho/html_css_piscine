# Coding Standard

모든 HTML & CSS 코드는 Google이 제공하는 style guide에 맞춰 작성해야 합니다. 아래 명시한 두 기준은 예외적으로 허용합니다.

- 3.1.1 void element를 닫지 않아도 됩니다.
- 4.2.8 CSS에서 작은 따옴표, 큰 따옴표 모두 허용합니다.
